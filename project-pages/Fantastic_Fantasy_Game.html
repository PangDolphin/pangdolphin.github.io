Fantastic Fantasy Game
A Computer Science project that uses my knowledge of objected oriented programming to create a turn based game on python.
Programming Language: Python
Project Details
This project was created for my Year 11 Term 2 Computer Science Practical Project. It was created with our new experience and knowledge of Object Oriented Programming. The task was to develop and improve the game using efficient and modular code.
This project featued 11 unique characters, A big 2D map, Store system, Levelling system with exp levels, Gold system, Turn based fighting system, Landmines, Job system.
