Belt Fed Confetti Launcher
This is a robotics project designed to launch projectiles using flywheels and a rotational magazine for storing shells.
Project Details
This project was created in Year 11, Term 4 for my robotics project. This project utilizes 2 digital inputs, 2 analogue inputs and 4 digital outputs and 1 analog input. Main features consist of a manual firing and reloading button, 2 potentiometers to control the flywheels and automatic firing mode. The automatic firing mode uses an analog to digital conversion for an on/off state.
The main components of this project is 1 NEMA 17, 2 L298n, 2 HUSETOO-775, 2 Button, 2 Potentiometer, 1 Solenoid Actuator, 1 Arduino, 1 Tip31C.
