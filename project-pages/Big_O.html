Big O | Time Complexity Simulation
A Computer science project that uses different sorting and searching algorithms to generate a graphs that represent the Big O.
Project Details
This project was to show the different time complexities inside of each algorithm. This is achieved by simulating each input with randomised data, which gets sorted and searched.
The time Complexities of each algorithm was generated onto a graph.
